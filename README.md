# 非アイランドスタイルeFPGA向けSAベース配置・配線ツール

本リポジトリには、非アイランドスタイルeFPGAを対象とした配置・配線を行うスクリプトが含まれています。

本コードは **FPGA設計フローの自動化技術（EDA）に関する研究の一環として筆者が実装したものです。**

本リポジトリは就職活動におけるポートフォリオとして限定公開しているものであり、
入力には研究チーム内で使用している独自形式のネットリストを用いるため、
第三者がそのまま実行・再利用することはできません。

---

# 背景と概要

一般的なeFPGAは **アイランドスタイルの配線構造** を採用しており、
柔軟な配線が可能である一方で、多数のMUXを必要とするため
**構成メモリ量が大きくなる**という課題があります。

本研究チームでは、この課題を解決するために
**非アイランドスタイルeFPGA** という配線アーキテクチャを提案しています。

この構造では、論理セル（PAE Cell）間に専用配線を導入し、
配線構造に制約を設けることで構成メモリの削減を目指します。

しかし、このようなアーキテクチャは既存のFPGA CADツールでは
配置・配線を行うことができません。

そこで本研究では、**Simulated Annealing（焼きなまし法）を用いた配置・配線最適化アルゴリズム** を実装し、
非アイランドスタイルeFPGAにおける配置・配線の自動化を行っています。

本ツールでは、配置と配線を独立した工程として分離するのではなく、
**配線制約を考慮しながら配置・配線を同時に最適化**することで、
アーキテクチャ制約を満たす実装を探索します。

---

# アルゴリズム構造

本ツールでは Simulated Annealing を用いて
配置・配線解の探索および遅延最適化を行います。

## Phase 1：配線制約を満たす配置・配線解の探索

まず、アーキテクチャ固有の配線制約を考慮しながら
**配線制約に矛盾しない配置・配線解** を探索します。

Simulated Annealing によってセル配置を更新しながら、
その配置において回路が配線可能かを評価し、
実装可能な配置・配線解を探索します。

---

## Phase 2：クリティカルパス遅延の最適化

Phase 1 によって実装可能な配置・配線解が得られた後、
回路の **クリティカルパス遅延の最適化** を行います。

評価では以下を考慮します。

* クリティカルパスの抽出
* MUX入力数に基づく配線遅延推定
* セル内部遅延

これらを評価関数に組み込み、
Simulated Annealing によって回路遅延の改善を行います。

---

# スクリプト構成

本リポジトリには、SAベースの配置・配線アルゴリズムを実装した
以下のスクリプトが含まれています。

## SimulatedAnnealingSparse.py

非アイランドスタイルeFPGAにおける配置・配線探索アルゴリズムを実装したメインスクリプトです。
Simulated Annealing により、配線制約を満たす配置・配線解の探索および遅延改善を行います。

## SimulatedAnnealingSparse_FF1.py

SimulatedAnnealingSparse.py をベースに拡張したスクリプトです。
4入力3出力の論理セル構造において、出力3本に対してFF数を1個に制限したアーキテクチャに対応しています。

---

# 本スクリプトの研究における位置づけ

本スクリプトは、EDA研究チームにおいて
**非アイランドスタイルeFPGAに機能回路を論理実装するためのツールの一部**として使用されています。

研究チームでは主に以下のテーマが並行して研究されています。

* テクノロジマッピング
* 論理セル構造の設計
* 配線アーキテクチャの設計
* 配置・配線アルゴリズム

本リポジトリのスクリプトは、その中で
**配置・配線ツールの役割** を担っています。

なお、ここで扱う「配置・配線」は
LSIの物理レイアウト設計ではなく、

**eFPGAアーキテクチャ上に回路機能を論理実装するための配置・配線**

を指します。

---

# eFPGA論理実装フロー

eFPGAに回路を実装する際の基本的なフローは以下の通りです。

HDL回路
↓
論理合成
↓
テクノロジマッピング
↓
配置・配線 ← 本リポジトリのツール
↓
Bitstream生成
↓
eFPGA上での動作

本ツールはこのフローの中で
**配置・配線工程を担当するEDAツール**として使用されます。
